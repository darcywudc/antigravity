Chapter 15

抽象向量空间
Abstract Vector Spaces

到目前为止，我们一直把向量想象成带有坐标的箭头。但线性代数的真正力量在于： 任何满足特定规则的对象都可以是"向量"。 函数是向量，多项式是向量，甚至声音信号也是向量。这是本系列的终极抽象。

15.1 超越箭头

15.1.1 向量的本质

回想一下我们对向量做过的操作：

  • 加法：两个向量相加得到一个新向量
  • 数乘：用标量缩放一个向量

这就是向量的全部本质。任何能做这两件事的对象集合，都可以叫做向量空间。

15.2 向量空间的公理

正式地，一个向量空间必须满足 8 条公理：

加法公理
  1. $\vec{u} + \vec{v} = \vec{v} + \vec{u}$（交换律）
  2. $(\vec{u} + \vec{v}) + \vec{w} = \vec{u} + (\vec{v} + \vec{w})$（结合律）
  3. 存在零向量 $\vec{0}$
  4. 每个向量有逆元 $-\vec{v}$
数乘公理
  1. $c(d\vec{v}) = (cd)\vec{v}$
  2. $1 \cdot \vec{v} = \vec{v}$
  3. $c(\vec{u} + \vec{v}) = c\vec{u} + c\vec{v}$
  4. $(c + d)\vec{v} = c\vec{v} + d\vec{v}$

这些规则看起来"显然"，但它们的力量在于：任何满足这些规则的系统，所有线性代数的理论都自动适用！

15.3 令人惊奇的例子

函数空间

所有从 $\mathbb{R}$ 到 $\mathbb{R}$ 的函数构成向量空间。

  • 加法：$(f + g)(x) = f(x) + g(x)$
  • 数乘：$(cf)(x) = c \cdot f(x)$
多项式空间

所有次数 ≤ n 的多项式构成向量空间 $P_n$。

基：$\{1, x, x^2, \ldots, x^n\}$
维度：$n + 1$

矩阵空间

所有 m×n 矩阵构成向量空间。

维度：$m \times n$

信号空间

音频信号可以看作函数向量。 傅里叶变换是换基操作！

15.4 抽象的线性变换

在抽象向量空间中，线性变换的定义不变：

$$ T(\vec{u} + \vec{v}) = T(\vec{u}) + T(\vec{v}) \quad,\quad T(c\vec{v}) = cT(\vec{v}) $$

例子：微分算子

$D: P_n \to P_{n-1}$，定义为 $D(p) = p'$（求导）。

这是线性的！$(f + g)' = f' + g'$ 且 $(cf)' = cf'$。

微分可以表示成矩阵！如果基是 $\{1, x, x^2, x^3\}$：

$$ D = \begin{bmatrix} 0 & 1 & 0 & 0 \\ 0 & 0 & 2 & 0 \\ 0 & 0 & 0 & 3 \\ 0 & 0 & 0 & 0 \end{bmatrix} $$

15.5 大图景

线性代数的普适性

不论向量是箭头、函数、还是多项式——只要满足向量空间公理， 我们学过的所有概念都适用：

  • 基与维度
  • 线性变换与矩阵
  • 行列式与逆
  • 特征值与特征向量

这就是抽象的力量：一套理论，无限应用。

本章总结 & 系列回顾

核心要点
  1. 向量空间：满足 8 条公理的任何对象集合。
  2. 函数、多项式、矩阵、信号都可以是向量。
  3. 线性变换（如微分）可以表示为矩阵。
  4. 所有线性代数概念在抽象空间中普适。
