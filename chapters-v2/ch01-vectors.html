🎯 1.1 从生活说起：标量 vs 向量

在日常生活中，有些量只需要一个数字就能完全描述：

例
只需要一个数字的量（标量）
  • 温度：25°C
  • 价格：100 元
  • 时间：3 小时
  • 重量：50 公斤

但还有些量，只说一个数字是不够的：

例
需要方向的量（向量）
  • 速度："向北 60 km/h"
  • 力："向右推 10 牛顿"
  • 位移："向东走 500 米"
  • 风："西北风 5 级"
定义：向量

向量 = 同时拥有大小（magnitude）和方向（direction）的量

用箭头表示：箭头的长度表示大小，指向表示方向。

📋 标量 vs 向量 对比表
场景 标量 (Scalar) 向量 (Vector)
🏃 跑步 速度 5 m/s 向东 5 m/s
💨 刮风 风速 10 m/s 西北风 10 m/s
🏗️ 推力 100 牛顿 向上 100 牛顿
🎮 游戏 生命值 100 位置 (50, 120)
⚠️ 常见误区：距离 vs 位移

距离是标量（只有大小），位移是向量（有方向）。

例如：绕操场跑一圈，距离是 400 米，但位移是 0（因为回到了起点）！

下列哪些是向量？哪些是标量？

  1. 飞机的航向
  2. 水的温度
  3. "往东北方向走 1.8 公里"
👆 点击查看答案
  1. 向量 — 航向有方向（如"向北 30°"）
  2. 标量 — 温度只有数值，没有方向
  3. 向量 — 有完整的方向信息 + 大小（1.8km）

👁️ 1.2 三种眼光看向量

不同领域的人，用不同的方式理解向量。这三种视角都对，而且相互补充：

🎮
游戏程序员的视角：坐标

向量 = 一组有序的数字，如 $(3, 4)$ 或 $(x, y, z)$

"角色在位置 (100, 200)，需要移动 (50, -30)"

📐
几何学家的视角：箭头

向量 = 一根有方向的箭头

"从原点指向某个位置的箭头"

🏗️
工程师的视角：物理量

向量 = 有方向的物理量（力、速度、位移...）

"结构受到 (10, -5) 千牛的外力"

💡 关键洞察

这三种视角描述的是同一个东西！
线性代数的威力在于：同一套数学工具可以处理游戏坐标、物理力、数据点...

✍️ 1.3 向量怎么写？

向量有几种常见的书写方式：

📝 常见表示法
箭头符号 $\vec{v}$ 或 $\overrightarrow{AB}$
粗体字母 $\mathbf{v}$（教材中常见）
列向量（竖着写） $\begin{bmatrix}3\\4\end{bmatrix}$
行向量（横着写） $(3, 4)$ 或 $[3, 4]$
约定

本课程中，默认使用列向量：$\vec{v} = \begin{bmatrix}x\\y\end{bmatrix}$

向量的分量 (components)：$x$ 叫做 x 分量，$y$ 叫做 y 分量。

➕ 1.4 向量加法：首尾相接

🎮 问题：游戏角色先向右走 3 步，再向上走 4 步，最终在哪里？

1
用向量表示两次移动

向右 3 步：$\vec{v} = \begin{bmatrix}3\\0\end{bmatrix}$

向上 4 步：$\vec{w} = \begin{bmatrix}0\\4\end{bmatrix}$

2
向量加法：对应分量相加
$\vec{v} + \vec{w} = \begin{bmatrix}3\\0\end{bmatrix} + \begin{bmatrix}0\\4\end{bmatrix} = \begin{bmatrix}3+0\\0+4\end{bmatrix} = \begin{bmatrix}3\\4\end{bmatrix}$
✓
答案

最终位置是 $(3, 4)$

向量加法公式
$\begin{bmatrix}a_1\\a_2\end{bmatrix} + \begin{bmatrix}b_1\\b_2\end{bmatrix} = \begin{bmatrix}a_1 + b_1\\a_2 + b_2\end{bmatrix}$

几何意义：将两个箭头首尾相接，从第一个箭头的起点到第二个箭头的终点。

💡 直觉理解

向量加法就像两段旅程的组合：
先走 $\vec{v}$，再走 $\vec{w}$，等于直接走 $\vec{v} + \vec{w}$。

� 向量加法图示

→ $\vec{v}$ + → $\vec{w}$ = → $\vec{v}+\vec{w}$

首尾相接：把 $\vec{w}$ 的起点放在 $\vec{v}$ 的终点，
从 $\vec{v}$ 的起点到 $\vec{w}$ 的终点就是和向量。

✖️ 1.5 数乘：缩放向量

🏗️ 问题：一个力的大小是另一个力的 2 倍，方向相同，怎么表示？

数乘（标量乘法）

用一个数字（标量）乘以向量的每个分量：

$c \cdot \begin{bmatrix}a_1\\a_2\end{bmatrix} = \begin{bmatrix}c \cdot a_1\\c \cdot a_2\end{bmatrix}$
例1
放大 2 倍

$2 \cdot \begin{bmatrix}3\\4\end{bmatrix} = \begin{bmatrix}6\\8\end{bmatrix}$

方向不变，长度变为 2 倍

例2
缩小到一半

$0.5 \cdot \begin{bmatrix}3\\4\end{bmatrix} = \begin{bmatrix}1.5\\2\end{bmatrix}$

方向不变，长度减半

例3
反向

$-1 \cdot \begin{bmatrix}3\\4\end{bmatrix} = \begin{bmatrix}-3\\-4\end{bmatrix}$

方向相反，长度不变

数乘的几何效果
$c > 1$ → 拉长
$0 < c < 1$ → 缩短
$c = 0$ → 变成零向量
$c < 0$ → 反向 + 缩放

📏 1.6 向量的长度（模）

向量的模 (Magnitude)

向量 $\vec{v} = \begin{bmatrix}x\\y\end{bmatrix}$ 的长度（模）记作 $|\vec{v}|$ 或 $\|\vec{v}\|$：

$|\vec{v}| = \sqrt{x^2 + y^2}$

这就是勾股定理！向量的长度 = 斜边长度。

例
计算 $\vec{v} = (3, 4)$ 的长度

$|\vec{v}| = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5$

📐 3D 向量的模

在三维空间中：

$|\vec{v}| = \sqrt{x^2 + y^2 + z^2}$

🎯 实战应用：用向量解决真实问题

🎮 问题 1：游戏角色路径计算

场景：游戏角色从 A$(2, 3)$ 走到 B$(8, 11)$，移动速度是 2 单位/秒。

问题：需要多长时间？方向向量是什么？

1
求位移向量

$\vec{AB} = B - A = (8-2, 11-3) = (6, 8)$

2
求距离（向量的模）

$|\vec{AB}| = \sqrt{6^2 + 8^2} = \sqrt{36+64} = \sqrt{100} = 10$

✓
答案

时间 = 距离 ÷ 速度 = 10 ÷ 2 = 5 秒

单位方向向量 = $\frac{1}{10}(6, 8) = (0.6, 0.8)$

🏗️ 问题 2：结构受力分析

场景：一个节点受到两个力：$\vec{F_1} = (30, 40)$ 牛顿，$\vec{F_2} = (-10, 20)$ 牛顿。

问题：合力大小是多少？

1
求合力（向量加法）

$\vec{F} = \vec{F_1} + \vec{F_2} = (30-10, 40+20) = (20, 60)$

✓
答案

$|\vec{F}| = \sqrt{20^2 + 60^2} = \sqrt{400+3600} = \sqrt{4000} \approx 63.2$ 牛顿

📊 问题 3：数据点的表示

场景：一名学生的成绩可以表示为向量 $(数学, 英语, 物理) = (85, 90, 78)$。

思考：这个"向量"的"长度"有意义吗？

✓
洞察

在数据科学中，向量不一定是"物理箭头"，但数学运算完全一样！

两个学生成绩向量的"距离"可以表示他们的相似程度。
