🔢 10.1 点积的定义

点积 (Dot Product)

两个向量的点积：

$\vec{a} \cdot \vec{b} = a_1 b_1 + a_2 b_2$

结果是一个数（标量），不是向量！

例
计算 $(3, 4) \cdot (1, 2)$

$= 3 \times 1 + 4 \times 2 = 3 + 8 = 11$

📐 10.2 几何意义：夹角与投影

点积与夹角
$\vec{a} \cdot \vec{b} = |\vec{a}| \cdot |\vec{b}| \cdot \cos\theta$

其中 $\theta$ 是两向量的夹角。

推论1
点积 > 0 → 夹角 < 90°（同向）
推论2
点积 = 0 → 夹角 = 90°（垂直）
推论3
点积 < 0 → 夹角> 90°（反向）
💡 核心直觉

点积测量的是两个向量的"方向相似度"。
完全同向 = 最大正值；完全反向 = 最大负值；垂直 = 0

📽️ 10.3 向量投影

向量投影

$\vec{a}$ 在 $\vec{b}$ 方向上的投影：

$\text{proj}_{\vec{b}}(\vec{a}) = \frac{\vec{a} \cdot \vec{b}}{|\vec{b}|^2} \vec{b}$
☀️ 类比：影子

想象阳光垂直照射 $\vec{b}$ 方向，$\vec{a}$ 的影子就是投影。
投影告诉你：$\vec{a}$ 在 $\vec{b}$ 方向上"有多少分量"。

🎯 实战应用

🎮 问题：视野检测

玩家面向 $(1, 0)$，敌人方向是 $(0.8, 0.6)$。敌人在视野内吗？

点积 = $1 \times 0.8 + 0 \times 0.6 = 0.8 > 0$ → 在前方 ✓

🏗️ 问题：力的分量

斜拉力 $\vec{F} = (100, 50)$ 牛顿，沿水平方向的分量是多少？

沿 $(1, 0)$ 方向投影 = $100 \times 1 + 50 \times 0 = 100$ 牛顿

📊 问题：余弦相似度

在推荐系统中，用户偏好向量的点积表示相似程度。

🎯 本章收获

核心要点
  1. 点积 = $a_1 b_1 + a_2 b_2$（结果是标量）
  2. 几何意义：$|\vec{a}||\vec{b}|\cos\theta$
  3. 点积 = 0 ⟺ 垂直（正交）
  4. 投影 = 向量在某方向上的"分量"
